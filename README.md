# 🧾 Billing App (React + Redux + Firebase)

A modern **Billing Application** built using React that allows users to browse products, add them to cart, view order summary, and place orders with Firebase integration.

---

## 🚀 Features

- 🛍️ Product Listing Page
- ➕ Add to Cart functionality
- 🛒 Cart Management (increase/decrease quantity)
- 💰 Order Summary with Discount Calculation
- 💳 Checkout & Place Order
- ☁️ Firebase Firestore Integration (Order Storage)
- 🎨 Clean & Responsive UI
